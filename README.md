Support Ticket Tracker (Python)
Overview
A command-line support ticket tracking tool that allows users to create, view, and update support tickets.

Features
Create support tickets with title and description
View all existing tickets
Update ticket status
Menu-driven CLI interface
Purpose
Built to practice Python fundamentals while modeling a real-world technical support workflow, including ticket creation, tracking, and resolution.

Skills Demonstrated
Python lists and dictionaries
Loops and conditional logic
User input validation
Clear, readable output formatting
How to run
Clone the repository
Run: python ticket_tracker.py
